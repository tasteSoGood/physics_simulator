6. 多体运动模拟 [[file:gravity.py][code]]

   这个demo比较特殊，多体运动的模拟如果不设置边界，质点很快就会消失，或者如果两个质点距离过近，速度会变得很快，这都不利于长时间观察系统的运动。

   这里考虑两点：1. 质点运动到边界时，类似小球碰撞到墙壁，会反弹回来；2. 质点速度超过一定阈值，则设置阻尼来降速。
